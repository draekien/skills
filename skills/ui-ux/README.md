# UI/UX

Skills for user interface and experience design: dashboards, visual design, and usability.

- [design-callouts](design-callouts/SKILL.md) — Applies callout design principles to design, build, or review callouts: status messages, docs admonitions, and attention/onboarding overlays, with component selection, accessibility requirements, and an anti-pattern catalog.
- [design-dashboards](design-dashboards/SKILL.md) — Applies dashboard design principles to design, build, or review dashboards: dashboard-type classification, chart-selection decision trees, KPI and layout rules, and an anti-pattern catalog.
