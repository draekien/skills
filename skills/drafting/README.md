# Drafting

Skills for writing and editing content: emails, documents, reports, and other written output.

- [draft-a-prd](draft-a-prd/SKILL.md) — Aligns with the user on scope and requirements, then produces a structured PRD document.
- [skill-evals](skill-evals/SKILL.md) — Builds and runs an empirical eval suite against a skill, measuring whether its description activates correctly and whether loading it improves agent output.
- [skill-writing](skill-writing/SKILL.md) — Creates, updates, and refines agent skills following the Agent Skills open standard.
- [vet-skill-idea](vet-skill-idea/SKILL.md) — Evaluates whether a concept should become a skill, gating it against worthiness criteria and routing it to build, split, redirect, or drop.
