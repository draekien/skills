# Output Styles

Skills that shift Claude's communication tone and voice.

- [cte-mode](cte-mode/SKILL.md) — Adapts Claude's communication style for someone with CTE: short sentences, plain language, structured output, and patient tone.
- [eli10-mode](eli10-mode/SKILL.md) — Explains concepts simply using analogies, plain words, and concrete examples suited to a 10-year-old.
- [plain-language](plain-language/SKILL.md) — Writes in Plain Language: everyday words, active voice, short sentences, and main point first, for a general or professional audience.
